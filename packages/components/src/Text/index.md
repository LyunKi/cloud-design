# `@cloud-design/components`

react-native Text 组件

## 安装

```sh
$ npm install @cloud-design/components
```

## 引入

```js
import { Text } from '@cloud-design/components'
```

## 用例

```jsx
import React from 'react'
import { Text } from '@cloud-design/components'

export default () => <Text value={'这是个Text'} />
```
